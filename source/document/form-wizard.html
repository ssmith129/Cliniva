get_app Installation

To use the Form Wizard (Stepper), you need to import MatStepperModule in your module. 

import {MatStepperModule} from '@angular/material/stepper';

@NgModule({
  imports: [
    MatStepperModule
  ],
})
export class YourModule { }

code Basic Usage

The stepper consists of mat-horizontal-stepper or mat-vertical-stepper with mat-step elements. 

<mat-horizontal-stepper>
  <mat-step label="Step 1">
    Content 1
  </mat-step>
  <mat-step label="Step 2">
    Content 2
  </mat-step>
</mat-horizontal-stepper>
